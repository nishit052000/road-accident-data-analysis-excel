🔹 Project Title
Road Accident Data Analysis Dashboard (Excel)

🔹 Project Overview
This project focuses on analyzing road accident data to identify key patterns, trends, and risk factors. The goal is to transform raw data into meaningful insights using Excel and present them through an interactive dashboard.

🔹 Tools Used
Microsoft Excel
Pivot Tables
Charts & Slicers

🔹 Key Features
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Interactive dashboard with dynamic filtering
KPI tracking (total accidents, casualties, severity)

🔹 Key Insights
Identified peak accident periods based on time and season
Highlighted high-risk locations with maximum accidents
Analyzed accident severity distribution
